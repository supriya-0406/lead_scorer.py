# lead_scorer.py

 A lightweight, rule-based lead scoring prototype that ingests B2B lead data and outputs conversion probability scores with recommended actions.

## ✨ Features
- ✅ Handles mixed data types (categorical, numerical, boolean)
- ✅ Rule-based intent scoring with recency decay
- ✅ Outputs ranked leads with actionable recommendations
- ✅ Easy to extend with ML models (XGBoost placeholder included)
